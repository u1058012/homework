策展主題說明






    
	
  1. 為感謝台灣捐贈口罩給日本，及對311東日本大地震的關懷與協助，日本當代藝術家奈良美智（YOSHITOMO NARA）於今年3月起在台展開的《奈良美智特展》巡迴展，精彩最終站即將在臺南市
    美術館一館登場，並以2021年初完成的《微熱少女Slight Fever》為主視覺，邀請民眾一同見證奈良創作的成長與變化。
    2. 
    
  2. 「奈良美智特展」由世界知名藝術家奈良美智親自帶來最新繪畫、逾十件素描新作，以及他創作歷程中最具代表性的繪畫、素描及雕塑作品。
    3. 
    
  3. 關渡美術館及高雄市立美術館展出後，臺北站53件作品、高雄站首度公開的素描作品、繪於紙板上的《山子姊姊》（Yamako Sister/older）與《山子妹妹》（Yamako Sister/younger）、
    奈良美智親自拍攝的《旅行的山子》（Traveling Yamako）等作品皆全數巡迴至壓軸場臺南站。
    4.     



 



 






注意事項







    
	
  1. 為尊重智慧財產權，本展覽「全面禁止攝錄影」。
    2. 
    
  2. 入館參觀請全程配戴口罩，並保持安全社交距離。
    3. 
    
  3. 展場內，禁止破壞及碰觸展品。如因人為造成損壞，需負相關賠償責任。
    4. 
    
  4. 本館展場內禁止飲食、吸菸，並禁帶食物、飲料入場參觀，如攜帶物件超過A3大小，請先至一樓寄物櫃或一樓服務台寄放。
    5.
